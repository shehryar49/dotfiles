# Dotfiles for my arch setup
This repo contains config files for Hyprland + Wofi + Waybar + Neovim
Everything should work by simply copying except the nvim folder, it requres 'thegentleman' folder to be renamed to your username
## Preview
It's a minmalistic config.

## How to install
```
for f in *; do cp -r $f ~/.config/; done
```
